kremalicious3
==================

[kremalicious.com](http://kremalicious.com) based on Jekyll

Requirements
------------------

- [NPM](https://npmjs.org/)
- [Bower](http://bower.io/)
- [Bundler](http://bundler.io/)

Get up and running
------------------

Install dependencies
```bash 
npm install && bower install && bundle install
```

Local development
```bash
grunt server
```

Production build
```bash
`grunt build`
```
